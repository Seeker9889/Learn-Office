Short-cut Keys

When the chart says Some button + some other button press the given buttons at the same time.

For example, "Ctrl+z" means press the Control button and the Z button at the same time.

Universal
Ctrl+s Save Save the current document (If this is the first save, it will instead be "Saved As" and you will have to choose a name for the document)
Ctrl+z Undo Undo the last change you made to the document/presentation/whatever
Ctrl+y Redo Redo the last change on which you used undo
Ctrl+x Cut Delete the selection and save it temporarily to be pasted in.
Ctrl+c Copy Temporarily save the selection so that it can be pasted in elsewhere.
Ctrl+v Paste Paste the selection
Ctrl+a Select all Select the entire document/page
Ctrl+b Bold Make the selected text bold
Ctrl+i Italicize Italicize the selected text
Word
Ctrl+Enter Page Break Add a break to continue the document on the next page
